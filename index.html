v1.2.01 2026-03-14　Bug 修復

小修
🐛 Bug 修復
  • 修正首頁「今日出勤筆數、出勤人數、簽到人數」顯示 0 的問題
  • 修正簽退頁面「出勤次數」顯示 0 的問題
  • 根本原因：Google Sheets 日期以 UTC 儲存（如 2026-03-13T16:00:00.000Z），需加 +8 小時轉換為台灣時間才能正確比對當日日期
  • 統一所有日期解析邏輯為 normDate()，支援 ISO+T、斜線格式、中文格式、Date(timestamp) 等多種格式
🎨 UI 修正
  • 統一所有輸入欄位（input、select）高度，解決欄位高度不一致問題
  • 導覽列加入版本修改紀錄頁面，側邊欄支援上下滑動
  • 查詢頁服務項目下拉選單補齊全部 16 項
